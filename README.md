# Concert Vast Parser

Exploratory work into parsing Vast tags ourselves.

### Clone it and Run it
* Clone this repo
* Run `yarn install`
* Run `yarn test` to run Jest in watch mode.

### Contributing
* Make a branch
* Add failing tests
* Write code to statisy test
* Make a Pull Request

### Important details:
* es6
* no semicolons _(come at me)_
* Strict TDD via [Jest](https://jestjs.io/)
